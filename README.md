# MarsCDODNet: A Topography-Aware Two-Branch Spatiotemporal Network for Martian Dust Optical Depth Forecasting

This repository contains the code implementation for our preliminary study,“MarsCDODNet: A Topography-Aware Two-Branch Spatiotemporal Network for Martian Dust Optical Depth Forecasting.”
This is a topography-aware deep learning model that combines historical atmospheric fields with static terrain information to improve the forecasting of global Martian column dust optical depth (CDOD). The repository also includes several recurrent baseline models for comparison.
